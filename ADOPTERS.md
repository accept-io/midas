## Add your organization

If you are using MIDAS in production or running a serious pilot, we would welcome a pull request to add your name here. This helps the project demonstrate real-world adoption and assists other organizations in their own evaluation.

To add your organization, open a PR that adds a row to the relevant table above. Include:
- Organization name (a link to your public site is welcome)
- A brief description of your use case (optional but appreciated)
- The approximate date you started using MIDAS

If you would prefer not to be named publicly, you can let us know via the [security contact](SECURITY.md) or open an anonymous issue — we will respect your preference.
