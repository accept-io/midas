# Maintainers

MIDAS is developed and maintained by Accept Labs.

## Current Maintainers

## Responsibilities

Maintainers are responsible for:
- Reviewing and merging pull requests
- Managing issues and discussions
- Setting project direction and roadmap
- Ensuring code quality and consistency

## Governance

MIDAS is currently maintained by two primary maintainers. As the project grows, additional maintainers will be added based on sustained contribution and community involvement.

Decisions are made transparently via GitHub issues and pull requests.
