# Authority Model

> ⚠️ **Coming Soon** — detailed documentation for this concept is planned for a future release.
> For now, see [docs/architecture/architecture.md](../architecture/architecture.md) for an overview
> of surfaces, profiles, grants, and the authority chain.

MIDAS evaluates whether a given actor is authorised to act on a given decision surface within a defined operational envelope. Authority flows in one direction: DecisionSurface → AuthorityProfile → AuthorityGrant → Agent.
